All notable changes to this project will be documented in this file. We follow the [Keep a Changelog](https://keepachangelog.com/) format.

### 🚀 v0.2.0 (2026-07-01) - Beta Release
*A major milestone, introducing core exchange features and Monero integration.*

#### ✨ Added
*   Implemented basic user profile management and neighbor search using geolocation APIs.
*   Integrated initial client-side functionality for encrypted chat messaging.
*   Added support for basic skill posting (Skill A $\leftrightarrow$ Skill B).
*   Initial connection to the MongoDB backend database structure.

#### 🛠 Changed
*   Updated API endpoints for better performance and security validation across the board.
*   Refined UI/UX flow for user onboarding and profile setup.

#### 🐛 Fixed
*   Fixed a bug where users could not see listings outside their immediate radius due to incorrect coordinate fetching.
*   Resolved issues with initial connection string loading on the backend server startup.

#### 🛡 Security
*   Implemented basic rate limiting on key public endpoints (login, registration) to mitigate brute force attacks.
